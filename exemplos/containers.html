Tags de Containers

Exemplos das principais tags de containers em HTML.

<section>

Esta é uma seção de conteúdo. Use-a para agrupar partes relacionadas de uma página.

<article>

Este é um artigo independente e reutilizável. Pode ser usado para posts de blog, notícias, etc.

<aside>

Este é um conteúdo lateral ou complementar, como uma barra lateral ou widget.

<nav>

Este é um menu de navegação. Use-o para agrupar links de navegação.

  • Home
  • Sobre
  • Contato

<div>

Este é um container genérico. Use-o quando nenhuma tag semântica for adequada.

<header>

Este é o cabeçalho de uma página ou seção. Pode conter títulos, menus ou logotipos.

<footer>

Este é o rodapé da página. Geralmente contém informações como copyright e links.

<fieldset>

Este container é usado para agrupar elementos de um formulário.
